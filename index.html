Basic Git Operations

Branch Management

gitmate "create a new branch for the login feature"
gitmate "switch to the main branch"
gitmate "delete the old-feature branch"

Committing Changes

gitmate "commit all changes with message 'fix login bug'"
gitmate "undo last commit but keep changes"
gitmate "commit only the src/index.js file"

Advanced Workflows

Merge & Rebase

gitmate "merge feature-branch into main"
gitmate "rebase current branch onto main"
gitmate "resolve merge conflicts"

GitHub Integration

gitmate "create pull request from current branch"
gitmate "list open pull requests"
gitmate "sync fork with upstream"
