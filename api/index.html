COMMANDS

Powerful Command Set

Use these commands and natural language to control your entire Git workflow.

Repository Commands

  • gitmate repo create <name> [--private] [--description "desc"]
  • gitmate repo list [--type=owner]

Git Commands

  • gitmate git init
  • gitmate git status
  • gitmate git add [files]
  • gitmate git commit <message>
  • gitmate push [branch]
  • gitmate pull [branch]
  • gitmate git revert last commit
  • gitmate git checkout <branch>
  • gitmate git branch

AI/NLP Features

  • Use natural language: gitmate push my changes to main
  • Generate .gitignore: gitmate generate gitignore for a Node.js project
  • Generate commit message: gitmate suggest a commit message
  • Create pull/merge requests: gitmate open a pull request from feature-x to main
  • Switch AI provider: gitmate ai switch to openai
  • Logout: gitmate logout

Security & Token Management

  • All tokens are encrypted and never shown in logs.
  • Supports secure login and logout flows.
  • Automatic token refresh and error handling.
