Installation

Get Started in Seconds

GitBot Assistant is distributed as a global npm package for easy installation and updates.

Single Command Setup

Install with npm and start using immediately with zero configuration.

Automatic Updates

Get new features and improvements with simple package updates.

Cross-Platform

Works on macOS, Windows, and Linux with consistent behavior.

terminal
npm install -g gitbot-assistant

First-time setup:

gitbot init
# Follow the interactive setup wizard
# Configure your AI provider and GitHub credentials

Run your first command:

gitbot "what can you do?"
Usage

How GitBot Works

Transform your natural language requests into precise Git operations

1

Speak Naturally

Describe what you want to do in plain English. GitBot understands your intent. 

gitbot "push my changes to main"
2

Interactive Confirmation

GitBot explains what it will do and asks for confirmation before proceeding. 

I'll push your changes to 'main'.
✔ Staged changes: 2 files
✔ Commit message: "Update config"
Proceed? (Y/n)
3

Execution & Results

GitBot executes the operation and provides clear feedback on the results. 

✓ Pushed to origin/main
✔ Commit: a1b2c3d
✔ Branch is up to date

Real-world examples of how GitBot can transform your workflow

Feature Branch Workflow

# Create and switch to new branch
gitbot "create branch feature-user-auth"

# Make changes and commit
gitbot "commit with message 'add user authentication'"

# Push to remote and create PR
gitbot "push and create pull request"

Repository Setup

# Create new private repository
gitbot "create private repo my-project"

# Initialize with README and license
gitbot "add readme and MIT license"

# Set up CI workflow
gitbot "add GitHub Actions workflow"

Complex Git Operation

# Safely undo last commit
gitbot "undo last commit but keep changes"

# Create backup branch
gitbot "create backup branch before merging"

# Merge with conflict resolution
gitbot "merge feature-branch with auto-resolve"

Bulk Operations

# Stage multiple files
gitbot "add all changed config files"

# Commit with generated message
gitbot "commit all with AI message"

# Push to multiple remotes
gitbot "push to origin and backup"
