GitMate Documentation

GitMate is an AI-powered Git assistant that transforms natural language into powerful version control operations. Instead of memorizing complex Git commands, simply tell GitMate what you want to accomplish in plain English.

Quick Start

Install GitMate and start using natural language Git commands in minutes:

npm install -g gitmate
gitmate init
gitmate "push my changes"

Installation

Prerequisites

  • Node.js 16+ installed on your system
  • Git configured on your machine
  • Internet connection for AI features
  • GitHub account (optional, for repository management)

Install GitMate

npm install -g gitmate

This installs GitMate globally on your system, making it available from any directory.

Verify Installation

gitmate --version

You should see the GitMate version number if the installation was successful.

Configuration

Initial Setup

gitmate init

This interactive command will guide you through the initial configuration:

  • Select your preferred AI provider (Mistral, OpenAI, or Anthropic)
  • Configure your API keys
  • Set up GitHub authentication (optional)
  • Configure default settings

Environment Variables

You can also configure GitMate using environment variables:

export AI_PROVIDER=mistral
export MISTRAL_API_KEY=your_api_key_here
export GITHUB_TOKEN=your_github_token_here

GitHub Authentication

gitmate auth github

This will open your browser for GitHub OAuth authentication and store your token securely.

Natural Language Commands

How It Works

GitMate uses advanced AI to understand your intent and execute the appropriate Git operations. Simply describe what you want to do in natural language:

Instead of:

git add .
git commit -m "fix authentication bug"
git push origin feature-branch

Just say:

gitmate "push my changes"

Command Examples

Basic Operations:
gitmate "push my changes"
gitmate "commit with message 'fix authentication bug'"
gitmate "show status"
Branch Management:
gitmate "create a new branch called feature-dashboard"
gitmate "switch to main branch"
gitmate "list all branches"
GitHub Operations:
gitmate "list all of my repos"
gitmate "create new repository my-project"
gitmate "who am i"

Git Operations

Repository Management

Initialize Repository:
gitmate "initialize git repository"
Clone Repository:
gitmate "clone repository from https://github.com/user/repo"
Add Remote:
gitmate "add remote origin https://github.com/user/repo.git"

Branch Operations

Create Branch:
gitmate "create branch feature-payment"
Switch Branch:
gitmate "checkout main branch"
Merge Branch:
gitmate "merge feature branch into main"

Advanced Git Operations

Revert Commit:
gitmate "revert last commit"
Show Diff:
gitmate "show differences"
View Log:
gitmate "show commit history"

GitHub Integration

Repository Management

List Repositories:

Create Repository:
gitmate "create new repository my-awesome-project"
User Information:

Pull Requests

Create Pull Request:
gitmate "create merge request from feature-dashboard to main"

GitMate will automatically check if your branch is ready, help you commit any pending changes, and guide you through the pull request creation process.

AI-Powered Features

Smart Commit Messages

gitmate "generate commit message"

GitMate analyzes your changes and suggests a conventional commit message based on what you've actually modified.

Intelligent .gitignore

gitmate "generate gitignore for Node.js project"

GitMate creates a comprehensive .gitignore file tailored to your project type and technology stack.

Context-Aware Operations

GitMate understands context and provides intelligent suggestions:

  • Automatically detects uncommitted changes before pushing
  • Suggests appropriate commit messages based on file changes
  • Warns about potential conflicts or dangerous operations
  • Provides helpful error messages and solutions

Advanced Features

Configuration Management

Configure Git User:
gitmate "configure git user"
Show Git Config:
gitmate "show git config"
Switch AI Provider:
gitmate "switch ai provider to openai"

Safety Features

Confirmation Prompts

GitMate asks for confirmation before executing destructive operations like force pushes or branch deletions.

Error Recovery

Provides helpful error messages and suggests solutions for common Git issues.

Troubleshooting

Common Issues

Authentication Errors

If you encounter authentication issues:

gitmate auth github
# Follow the browser authentication flow

AI Service Unavailable

If AI features are not working:

gitmate init
# Reconfigure your AI provider and API keys

Git Repository Issues

If GitMate can't find your repository:

gitmate "initialize git repository"
# This will create a new Git repository if needed
